Blinking LED
Goal

Make the LED blink using Arduino digital output.

Components
  • USB cable
  • Breadboard
  • Arduino UNO board
  • 1 × Yellow LED
  • Jumper wires
  • 1 × 220Ω resistor
Circuit explanation

The red wire is connected to pin 13 on the Arduino UNO board. The black wire is connected to GND on the UNO board. Current flows from pin 13 through the resistor, which is connected to the longer leg of the LED (anode). Then it passes through the LED and exits through the shorter leg (cathode), finally returning to GND through the black wire.

Code
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}
