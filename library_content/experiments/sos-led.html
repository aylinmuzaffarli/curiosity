SOS LED
Goal

Make an LED blink the SOS Morse code signal using Arduino digital output and timing control.

Components
  • USB cable
  • Breadboard
  • Arduino UNO board
  • 1 × Red LED
  • Jumper wires
  • 1 × 220Ω resistor
Circuit explanation

The Arduino digital pin 13 provides a HIGH signal that sends current through a 220Ω resistor to limit the current and protect the LED. The current then flows into the longer leg of the LED (anode), allowing it to light up. It exits through the shorter leg (cathode) and returns to the GND pin on the Arduino, completing the circuit.

Code
int unit = 200;
int pin = 13;

void setup() {
  pinMode(pin, OUTPUT);
}

void loop() {
  // S
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);
  delay(3 * unit);

  // O
  digitalWrite(pin, HIGH);
  delay(3 * unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(3 * unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(3 * unit);
  digitalWrite(pin, LOW);
  delay(3 * unit);

  // S
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);
  delay(unit);
  digitalWrite(pin, HIGH);
  delay(unit);
  digitalWrite(pin, LOW);

  delay(1000);
}

Morse code is built from one basic time unit. In this code, unit = 200 ms.

A dot lasts for 1 unit, and a dash lasts for 3 units. The pauses are also part of the code: 1 unit between parts of the same letter, 3 units between letters, and 7 units between words.

  • Dot = 1 × unit
  • Dash = 3 × unit
  • Gap inside a letter = 1 × unit
  • Gap between letters = 3 × unit

This is why the code uses delay(unit) and delay(3 * unit) — they control rhythm.

So the signal becomes ... --- ... : three short blinks, three long blinks, then three short blinks again.
