LEGO Rebuilt as a Case for an Arduino Reaction Game
Overview

I have built an Arduino-based reaction game. This game starts with a yellow LED: it randomly lights up, and the user should press the button as soon as possible. The system will blink green LED if the user was fast enough, and red otherwise.

Why I use a power bank and a physical case

I searched for options to walk with my projects. When the USB cable is plugged into my computer, I cannot do that. Until I discovered that it is possible with a portable power supply. I found my dad’s power bank and connected it to my project. My dad did not use it, so it is now mine :)

I built a physical case, since I noticed that it is really uncomfortable to carry all of these 3 pieces in my hands: power bank, Arduino UNO board and breadboard. So I thought: what if I could build a multi-layer case for carrying all of it? I have been thinking of cardboards, hot glue guns etc. Finally, I chose to do it with LEGOs, because they are easily editable. I didn't have LEGO except for my rocket (which I built during the Artemis 2 mission’s time).

A previous LEGO build that I repurposed for the case.

I have decided to break it, and improvise with what I have.

Finally, I've built what you see above. I did not expect it to be THIS good. It keeps my circuit secure and firmly fixed, so it doesn't move or loosen when the device is shaken.

I’m genuinely happy with it.

It is a prototype, version 1.

Components
  • Arduino UNO board
  • Breadboard
  • Power bank (enables mobility)
  • LEGO pieces (custom enclosure)
  • Push button
  • 1 × Red LED
  • 1 × Green LED
  • 1 × Yellow LED
  • Jumper wires
Code
int button = 2;


int red_led = 10;
int green_led = 11;
int yellow_led = 12;


unsigned long starttime;
unsigned long reactiontime;


void setup() {
  pinMode(button, INPUT_PULLUP);
  Serial.begin(9600);


  pinMode(red_led, OUTPUT);
  pinMode(green_led, OUTPUT);
  pinMode(yellow_led, OUTPUT);




  randomSeed(analogRead(A0));
}


void loop() {
   digitalWrite(red_led, LOW);
  digitalWrite(green_led, LOW);
  digitalWrite(yellow_led, LOW);




  delay(random(2000, 5000));
  digitalWrite(yellow_led, HIGH);
 
  starttime = millis();
  while (digitalRead(button) == HIGH) {


  }


  reactiontime = millis() - starttime;
  digitalWrite(yellow_led, LOW);
 
   if (reactiontime < 500) {
    digitalWrite(green_led, HIGH);
    delay(200);
    digitalWrite(green_led, LOW);
     delay(200);
    digitalWrite(green_led, HIGH);
    delay(200);
    digitalWrite(green_led, LOW);
     delay(200);
    digitalWrite(green_led, HIGH);
    delay(200);
    digitalWrite(green_led, LOW);
     delay(200);
   }
   else {
    digitalWrite(red_led, HIGH);
    delay(200);
    digitalWrite(red_led, LOW);
     delay(200);
    digitalWrite(red_led, HIGH);
    delay(200);
    digitalWrite(red_led, LOW);
 delay(200);

    digitalWrite(red_led, LOW);

   }
}
